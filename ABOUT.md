## Description (pour GitHub "About")

AstroGraphAnomaly: few-label anomaly detection on astronomical catalogs (Gaia), graph-based feature extraction, anomaly engines (IsolationForest/LOF/OCSVM/Robust), explainability (LIME), and systematic LLM prompt generation. Optional ingestion support for Hubble/HST catalogs via MAST (astroquery), depending on availability.
